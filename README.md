# Welcome to my home lab repository!

In this repo, I'll be recording my home lab journey. You can expect to find things like:

- Setup guides
- Configuration templates
- Concept diagrams

Stay tuned for more updates!

## Project Overview

### Phase 0: Basic Setup

- Install OS
- Enable SSH (key-only)
- Install command-line tools
- Disable Wi-Fi and Bluetooth
- Configure basic firewall

### Phase 1: Home Lab Foundation

- Establish ```/srv``` directory hierarchy
- Install and configure Docker
- Limit logging based on current storage
- Configure unattended security upgrades

### Phase 2 (IN PROGRESS): Basic Services

- Install Pi-hole (ad-blocker)
- Install Unbound (local DNS)
- Install Caddy (TLS and reverse proxy)
- Install Tailscale (WireGuard VPN)
- Secrets handling

### Phase 3 (FUTURE): Additional Services
